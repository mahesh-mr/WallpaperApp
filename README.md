# About

This is a simple wallpaper app, User can search images and download them on your mobile or set to wallpaper. Pixcels Api is used for this
# Features
- Search Images
- Pick categories
- Downlod Images
- Set to Homescreen
- Set to Lockscreen
- Set both Home& Lockscreem
